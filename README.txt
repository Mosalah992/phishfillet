PhishFillet - Advanced Email Security Analyzer
Version 1.0
=======================================

Description
-----------
PhishFillet is a powerful desktop application designed to analyze email files (.eml) for potential phishing threats and security concerns. With its modern, user-friendly interface and comprehensive analysis capabilities, it helps security professionals and users identify suspicious email characteristics and potential security risks.

Features
--------
1. Email Security Analysis
   - SPF (Sender Policy Framework) verification
   - DKIM (DomainKeys Identified Mail) authentication check
   - DMARC policy validation
   - Sender/Return-Path consistency check
   - Suspicious relay detection
   - Header chain analysis

2. Visual Indicators
   - Color-coded analysis results:
     * Red: High-risk issues
     * Orange: Medium-risk concerns
     * Green: Passed security checks
   - Special highlighting for:
     * IP addresses (light blue)
     * Domain names (green)
     * Email addresses (purple)

3. Header Analysis
   - Organized header information by categories:
     * Authentication
     * Routing
     * Sender Info
     * Recipient Info
     * Message Info

4. Export Capabilities
   - Export analysis results to CSV format
   - Detailed reporting of all findings

Installation
------------
1. Extract PhishFillet_v1.0.zip to your desired location
2. Ensure the 'images' folder remains in the same directory as the executable
3. Run PhishFillet.exe

System Requirements
-----------------
- Windows 10 or later
- 4GB RAM (minimum)
- 100MB free disk space
- Screen resolution: 1024x768 or higher

Usage
-----
1. Launch PhishFillet.exe
2. Click "Open Files" to select one or more .eml files
3. Review the analysis results in the main window
4. Use "Export CSV" to save the results for documentation

Known Issues
-----------
- Some email clients may save attachments separately from .eml files
- Very large email files (>10MB) may take longer to process
